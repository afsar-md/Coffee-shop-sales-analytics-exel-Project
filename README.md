# ☕ Coffee Shop Sales – Customer Analytics (Excel Project)
I developed a comprehensive project in excel , creating dashboard to analyze the data. This process involved several stages, including data preprocessing,data cleaning and data visulization.


## 📌 Project Overview
This project presents an **interactive Excel dashboard for Coffee Shop Sales Analytics**.  
The report analyzes customer behavior, sales performance, and product trends using key KPIs and visual charts.

The goal of this project is to **provide insights into customer purchasing patterns and improve business decision-making**.

## 🎯 Project Objective
The objective of this project is to analyze **customer behavior, sales performance, and product demand** using interactive Excel dashboards.

---

## 📊 Report Tabs

### 1️⃣ Customer Analytics
This tab provides a high-level summary of coffee shop sales using key performance indicators:

**Key KPIs**
- Total Sales  
- Total Footfall  
- Average Bill per Person  
- Average Orders per Person  

---

### 2️⃣ Chart Analysis
This tab contains visual charts to analyze sales and customer trends, including:

- Quantity Ordered by Hours  
- Category % Distribution Based on Sales  
- Size Distribution Based on Orders  
- Footfall and Sales by Store Location  
- Top 5 Products Based on Sales  
- Orders by Weekdays
